# DeveLanCacheUI_SteamDepotFinder_Runner
This runs the DeveLanCacheUI_SteamDepotFinder (https://github.com/devedse/DeveLanCacheUI_SteamDepotFinder) on a cron job and publishes the result.

| Project | Explanation |
